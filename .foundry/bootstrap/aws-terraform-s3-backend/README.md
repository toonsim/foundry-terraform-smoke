# AWS Terraform S3 Backend Bootstrap


This directory contains a reviewable AWS CLI bootstrap plan generated by Foundry.

The plan creates or updates:

- the GitHub Actions OIDC provider,
- the Terraform IAM role trust policy,
- a scoped inline policy for the configured Terraform state object and lockfile,
- the S3 state bucket with versioning, encryption, and public access blocking.

It does not create AWS access keys, write secret values, or attach administrator
policies. Review plan.sh and terraform-state-policy.json before running the
script with a bootstrap AWS CLI profile.
